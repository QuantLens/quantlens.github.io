From watchlist to winning trades in 3 steps

QuantLens delivers a complete market view with LLM‑powered TA.

1) Build your watchlist

Add assets (BTC/USDT, DXY, AAPL) in TradingView. New: start with 5; Pros: 50+ with multi‑chart layouts.

2) Set QuantLens alerts

Install the indicator, choose triggers (e.g., breakout, RSI>60), and create alerts on the study (Any alert(), once per bar close). Deliver via webhook or copy from Alert Log.

{
  "symbol": "DXY",
  "price": 100.50,
  "indicators": { "rsi": 52.26, "adx": 28.875, "vwap": 100.55 },
  "signals": { "breakout_long": { "trigger": 101.00, "stop": 100.35, "target": 101.50 } },
  "correlation": { "dxy_btc": -0.75, "context": "DXY weakness boosts BTC" }
}

3) Paste into your LLM

Feed the template to your LLM for a sophisticated TA report, then ask follow‑ups like “How does DXY’s trend affect BTC?”
