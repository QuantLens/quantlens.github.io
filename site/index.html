FAQ

Do you publish the underlying code or schema?

No. QuantLens provides outcomes, not internals. The implementation details and full schema are proprietary.

How do I try it?

Join the early access list. We provide a demo environment and sample snapshots so you can evaluate fit without accessing private code.

What markets are supported?

We focus on high‑liquidity instruments across crypto, equities, and index products. Coverage expands over time with user demand.

Can I integrate this with my AI stack?

Yes, with some setup required. QuantLens outputs structured JSON designed for AI consumption, but you'll need to build the webhook collector and processing pipeline. We provide sample code and guidance for safe integration without exposing proprietary internals, but this isn't a plug-and-play solution yet.

What's included in a snapshot?

Multi‑horizon indicators, volume profile metrics, support/resistance levels, signals with stops/targets, risk metrics, and market context — all in compact JSON format.
